## wu-ui-tools 工具集

> **组件名：wu-ui-tools**

wu-ui 工具集成，包括网络Http请求、便捷工具、节流防抖、对象操作、时间格式化、路由跳转、全局唯一标识符、规则校验等等。

需要在自己的项目中使用请参考[扩展配置](https://wu.geeks.ink/zh-CN/components/extendedConfiguration.html)。
